# 祝你生日快乐 - 本地页面

这是一个简单的本地静态页面，用于展示生日祝福和小组件（倒计时、蛋糕、气球和彩屑动画）。

使用方法：

1. 打开 `index.html`（在浏览器中双击或通过本地服务器打开）。
2. 在“名字”输入框中填写名字，页面会更新祝福标题。
3. 在“生日日期”中选择一个日期，会显示到最近一次生日的倒计时。
4. 点击「庆祝」按钮会触发气球和彩屑动画，以及蛋糕烛火闪烁。

文件：
- `index.html` - 页面主文件
- `styles.css` - 样式
- `script.js` - 交互逻辑（倒计时、气球、彩屑）

注意：为在本地获得最佳体验，请使用现代浏览器（Chrome/Edge/Firefox）。
